##### \## 🚀 Purpose \& Overview

Transitioning from game development to full-stack software engineering, I built this project to demonstrate my ability to make a website



\---



##### \## 🛠️ What I Built (Technical Architecture)



This website is a full-stack application leveraging a modern decoupled architecture:



###### \### 📺 Frontend (React)

\* \*\*Component-Driven UI:\*\* Recreated the official Nintendo layout using modular, reusable React components.

\* \*\*State Management:\*\* Handled complex UI states (such as active shopping carts, user authentication states, and store filters).

\* \*\*Responsive Design:\*\* Optimized the interface for seamless browsing across desktop, tablet, and mobile devices.



###### \### ⚙️ Backend (.NET)

\* \*\*RESTful API:\*\* Developed a robust .NET Web API to handle secure communications between the user interface and the database.

\* \*\*Database \& Business Logic:\*\* Implemented secure data handling for product catalogs, user accounts, and cart data.



\---



###### \## 🛠️ Tech Stack

\* \*\*Frontend:\*\* React, JavaScript/TypeScript, Tailwind CSS, Framer.motion

\* \*\*Backend:\*\* .NET Core API, C#

\* \*\*Database:\*\* MySql



\---



###### \## 📋 Features Demonstrated

\* Full user authentication and session management.

\* Dynamic product catalog with filtering and search capabilities.

\* Functional persistent shopping cart.

\* Smooth scrollable animations.
